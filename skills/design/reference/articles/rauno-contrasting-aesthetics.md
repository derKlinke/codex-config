## Reusable Principles

1. Contrast should be intentional, not random
   - Use contrast to create hierarchy or narrative emphasis.
   - Do not stack unrelated contrasts across every layer.

2. Keep invariants while introducing contrast
   - Hold core structure stable (grid, spacing rhythm, typographic cadence, or token semantics).
   - Vary one or two axes at a time (shape language, saturation, texture, motion character).

3. Contrast requires an anchor
   - A readable base aesthetic should dominate.
   - Contrasting elements should act as accents or focal deviations.

4. Tension must still preserve usability
   - Strong visual op  position is valid only when legibility, affordance clarity, and task flow remain intact.

5. Avoid novelty inflation
   - Repeated high-contrast treatment loses signal and increases noise.
   - Reserve strongest contrast for low-frequency/high-significance moments.
